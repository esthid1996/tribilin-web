## 📋 Descripción del Proyecto

IT Solutions es una página web profesional diseñada para presentar servicios de tecnología de información de alta calidad. El sitio ofrece una experiencia moderna, responsiva y atractiva para potenciales clientes.

### ✨ Características Principales

- **Diseño Responsivo:** Compatible con dispositivos móviles, tablets y escritorio
- **Moderno y Profesional:** Interfaz limpia con colores corporativos
- **Secciones Completas:**
  - Página de inicio (Hero section)
  - Catálogo de servicios
  - Información sobre la empresa y propietario
  - Portafolio de proyectos anteriores
  - Formulario de contacto
  - Footer informativo

- **Funcionalidades Interactivas:**
  - Menú hamburguesa responsive
  - Smooth scrolling entre secciones
  - Animaciones al cargar elementos
  - Efectos hover en tarjetas
  - Formulario de contacto funcional

## 🚀 Cómo Usar

### Abrir en Navegador
1. Navega a: `C:\Users\ESTHID\Desktop\Claude_proyecto\`
2. Haz doble clic en `index.html`
3. Se abrirá en tu navegador predeterminado

### Con Servidor Local
Para mejor funcionamiento, se recomienda usar un servidor local:

**Con Python:**
```bash
python -m http.server 8000
```

**Con Node.js (http-server):**
```bash
npx http-server
```

Luego accede a: `http://localhost:8000`

## 🔧 Personalización

### Cambiar Colores
Edita las variables CSS en `css/styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #10b981;
    /* ... más variables ... */
}
```

### Agregar Nuevas Secciones
1. Agrega la sección en `index.html`
2. Crea los estilos en `css/styles.css`
3. Agrega la funcionalidad en `js/script.js` si es necesario

### Modificar Contenido
- **Servicios:** Edita la sección `#servicios` en `index.html`
- **Portafolio:** Edita la sección `#portafolio` en `index.html`
- **Contacto:** Edita la sección `#contacto` en `index.html`
